Programmeringens Historia

Programmeringens historia börjar redan så tidigt som på 1800-talet med Ada Lovelace, som skrev den första algoritmen för en maskin. Programmeringen under denna tiden inkluderade inte elektroniska komponenter eftersom de inte fanns under denna tiden. Det var ett sätt att manipulera maskiner vilket på ett tekniskt sätt är programmering.

Det var inte förrän på 1940-talet som vi fick maskinkod för datorer. ENIAC var den första elektroniska datorn, vilket skapades för att knäcka krypterad data. Den programmerades genom att vrida på reglage och dra kablar på kopplingsbrädor. Detta kunde ta veckor. Data lästes sedan in från hålkort.

Under sent 1940-tal började man med assembler. Istället för att använda nollor och ettor kunde man nu använda assembler, vilket är ett programmeringsspråk på den lägsta nivån med hjälp av instruktioner. Ett lågnivåspråk kommunicerar direkt med komponenterna på datorn. Kathleen Booth utvecklade det första assembler-språket under 1947.

Ett av de första högnivåspråken var Fortran, vilket utvecklades under 1954. Ett högnivåspråk är ett programmeringsspråk där varje rad inte direkt är en instruktion i processorn utan måste översättas till maskinkod med hjälp av en kompilator. Fortran används än idag, bland annat för att testa hastigheten på superdatorer.

År 1959 skapades programmeringsspråket Cobol. Grace Hopper hade lagt grunden för ett plattformsoberoende programmeringsspråk under 50-talet. Språket hade en syntax som liknar engelska mer än matematiska formler. Detta ledde till utvecklingen av programmeringsspråket Cobol.

Under 1972 skapades ett av de kändaste och mest använda programmeringsspråken än idag, C. Språket utvecklades av Dennis Ritchie, som också jobbade på operativsystemet Unix tillsammans med en kollega. Mycket av grundkoden i operativsystem idag är fortfarande skriven i C på grund av optimering.

C++ är ett av de mest komplexa och mest kända programmeringsspråken. Språket utvecklades av Bjarne Stroustrup under 1985 och var ett tidigt exempel på objektorienterad programmering, vilket är ett sätt att organisera ett program med hjälp av objekt, som vi kallar för klasser.

År 1990 skapades ett av de mest simpla och kända programmeringsspråken, Python. Python används på många universitet för att lära ut programmering och av forskare och företag tack vare sina bibliotek. Men Python används sällan för att skapa en produkt eller tjänst. Python används mest som en prototyp för program.

Java är ett av de mest använda programmeringsspråken, och liknar en mix av andra språk. Skapad år 1995, Java är ett objektorienterat programmeringsspråk som C++, men har en unik modell och syntax där den kompilerade koden kan köras på olika processorer via Java Runtime Environment.
